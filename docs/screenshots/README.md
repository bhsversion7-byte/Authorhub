# README screenshots

These images appear in the **"A look inside"** section of the main [README](../../README.md).

They are currently **branded placeholders**. To show real captures, replace each file
**keeping the exact same filename** and commit:

| File | What to capture | Suggested size |
|---|---|---|
| `landing.png` | The landing page — the drifting women-writer quote orbit + the 3D book | ~1600×1000, landscape |
| `relation-graph.png` | A novel's relation graph with a few character "planets" and edges | ~1600×1000, landscape |
| `workspace.png` | A novel workspace — character cards / timeline / settings | ~1600×1000, landscape |

Tips:
- PNG or JPG both work (a short **.gif** is also fine for the landing to show the book animation — just keep it under ~5 MB so the README stays light).
- Capture at 100% zoom in light mode for the crispest look; a second dark-mode shot is welcome too.
- `.gitignore` already allows `docs/screenshots/*.png|jpg|gif`, so your files will be tracked.
